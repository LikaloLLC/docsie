# 보안 신뢰할 수 있는

보안 지식 베이스: 내부 및 외부 액세스

엔터프라이즈급 보안 기능으로 문서를 보호하세요

단일 로그인 (SSO)

Single Sign-On으로 보안을 강화하고 사용자 액세스를 간소화하세요. SSO는 로그인 프로세스를 단순화하여 패스워드 피로의 위험을 줄이고 문서화 시스템 전반에 걸쳐 사용자 인증을 강화합니다.

  • SAML 및 OAuth 통합
  • 중앙 집중식 사용자 관리
  • 패스워드 위험 감소

비밀번호 보호

강력한 비밀번호 보호를 통해 민감한 콘텐츠를 안전하게 보호하세요. 이 보안 계층은 승인된 사용자만이 중요한 정보에 액세스할 수 있도록 보장하여 무단 액세스로부터 데이터를 보호합니다.

  • 문서 수준 보호
  • 암호화된 데이터 저장소
  • 보안 공유 옵션

역할 기반 문서화

역할 기반 제어를 통해 콘텐츠 액세스를 사용자 정의하세요. 사용자 역할이나 클라이언트 요구사항에 따라 문서 가시성을 조정하여 모든 사용자가 자신에게 관련되고 적절한 콘텐츠에만 액세스할 수 있도록 보장합니다.

  • 세분화된 권한
  • 팀 및 클라이언트 액세스
  • 콘텐츠 표시 제어
